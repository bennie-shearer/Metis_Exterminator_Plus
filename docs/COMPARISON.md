# Comparison - Metis Exterminator Plus

**Version 2.3.0**

---

## vs. Enterprise SaaS (PestPac, ServSuite, Briostack, FieldRoutes)

| Feature | Metis Exterminator Plus | Enterprise SaaS |
|---|---|---|
| Cost | Free (MIT) | $100-$500/technician/month |
| Internet required | No | Yes |
| Data ownership | 100% yours | Vendor's servers |
| Setup time | Minutes | Hours to days |
| Customer portal | Roadmap | Yes |
| GPS route optimization | Roadmap | Yes |
| Mobile field app | Standalone HTML | Native app |
| Chemical tracking | Roadmap | Yes |
| QuickBooks integration | Roadmap | Yes |
| API access | Full REST included | Usually paid tier |
| Self-hosted | Yes | No |
| Source available | Yes (MIT) | No |

**When to choose Metis:** Single-owner or small operation, offline-first requirement, budget constraint, data sovereignty priority, technical user comfortable with self-hosting.

**When to choose enterprise SaaS:** Large operation (10+ technicians), need GPS dispatch, require regulatory compliance reporting out of the box, want a mobile native app, have an IT budget.

---

## vs. Spreadsheets (Excel, Google Sheets)

| Feature | Metis Exterminator Plus | Spreadsheets |
|---|---|---|
| Structured data | Yes (SQLite, normalized) | Flat, prone to drift |
| Search | Full-text search | Filter/sort only |
| Invoice numbering | Automatic sequential | Manual |
| Tax calculation | Automatic | Manual formula |
| Multi-user | Session-based auth | Google Sheets (online only) |
| Offline | Yes | Excel only |
| Backup | Single file copy | File save |
| Cost | Free | Free (Google) / $10+/mo (Microsoft) |

**When to choose Metis:** More than 20 customers, need invoicing, want structured search, have recurring jobs. **When to keep spreadsheets:** Fewer than 10 customers, pure scheduling only, no invoicing needed.

---

## vs. Paper / Manual Systems

Metis Exterminator Plus is an unambiguous upgrade from paper:
- Searchable customer records (no flipping through cards)
- Invoice history per customer instantly accessible
- Dashboard revenue summary without adding up receipts
- Backup in seconds (one file copy)
- Legible records (no handwriting issues)
- No physical filing space needed

---

## vs. Generic CRM (HubSpot Free, Zoho CRM Free)

Generic CRMs handle contacts and pipeline but lack pest-control-specific fields (pest type, service type, chemical application), job scheduling, invoice generation with tax, and service status tracking. Adapting a generic CRM to pest control requires significant customization and still lacks the domain-specific workflow. Metis is purpose-built.
